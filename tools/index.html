ROMi Database

ROM database files for ROMi (PS3 homebrew).

Installation

Choose one of the following methods to set up the databases on your PS3.

Option 1: Online Setup (Recommended)

With this method, ROMi will automatically download and refresh the combined database from the internet. This requires a working internet connection on your PS3.

  1. Create a new text file named config.txt on your computer.
  2. Add the following line to the file: 
    url https://matheusfillipe.github.io/ROMi/romi_db.tsv
  3. Copy your new config.txt file to your PS3 at:
    /dev_hdd0/game/ROMI00001/USRDIR/config.txt
  4. Start ROMi and select "Refresh" from the menu. The combined database file (containing all platforms) will be downloaded automatically.

Note: The database is downloaded once when you refresh. You can update it anytime by selecting "Refresh" again in the ROMi menu.

Option 2: Offline Setup

Use this method if you want to place all the database files on your PS3 manually and not rely on an internet connection for updates.

  1. Download the complete database archive:
    databases.zip
  2. Extract the .zip file.
  3. Copy all the extracted files (sources.txt and all romi_*.tsv files) to your PS3 at:
    /dev_hdd0/game/ROMI00001/USRDIR/
  4. Restart ROMi.

Database Files

The databases are updated weekly via GitHub Actions. The primary file for online updates is:

  • romi_db.tsv - Combined database (all platforms)

Individual platform database files (for offline use):

  • romi_GB.tsv - Game Boy
  • romi_GBC.tsv - Game Boy Color
  • romi_GBA.tsv - Game Boy Advance
  • romi_NES.tsv - NES
  • romi_SNES.tsv - SNES
  • romi_Genesis.tsv - Sega Genesis
  • romi_SMS.tsv - Sega Master System
  • romi_PSX.tsv - PlayStation
  • romi_N64.tsv - Nintendo 64
  • romi_Arcade.tsv - Arcade (MAME)
